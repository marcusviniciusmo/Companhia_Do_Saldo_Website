# English:

Design of a website for a clothing store selling fabrics in Fortaleza/CE, with a registration form for orders. It was developed with React technology on the Front-end using Typescript template. Concepts such as React Hooks, React Navigation, React Router Dom were also introduced, and React Icons. The Axios library was used to obtain the user's address on the order form.

---

# Português:

Criação de um site para loja de tecidos em Fortaleza/CE, com ficha cadastral de pedidos. Foi desenvolvido com tecnologia React no Front-end utilizando template Typescript. Conceitos como React Hooks, React Navigation, React Router Dom também foram introduzidos, e React Icons. Foi utilizada a biblioteca Axios para obter o endereço do usuário no formulário de
pedidos.
